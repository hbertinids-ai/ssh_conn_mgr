🚀 3-STEP INSTALLATION
══════════════════════════════════════════════════════════════

STEP 1: Open the App
────────────────────────────────────────────────────────────
  The app is already running at: http://localhost:5173
  
  (If not, run: npm run dev)


STEP 2: Open the Script File
────────────────────────────────────────────────────────────
  In VS Code, open:
  
    📄 scripts/populate-cat1-data.js
  
  Then:
    1. Select all (Ctrl+A)
    2. Copy (Ctrl+C)


STEP 3: Run in Browser Console
────────────────────────────────────────────────────────────
  In the app browser window:
    1. Press F12 (opens Developer Tools)
    2. Click "Console" tab
    3. Paste the script (Ctrl+V)
    4. Press Enter
    5. Press F5 to refresh the page

⚙️ NEXT STEPS
══════════════════════════════════════════════════════════════

1. CONFIGURE THE TUNNEL FIRST! ⚠️
   Go to Tunnels tab → CAT1 Connection group
   Click pencil (✏️) on "CAT Legacy Env Tunnel"
   Set: Host, Username, Password for your jump server
   Click Save

2. SET CONNECTION PASSWORDS
   Go to Connections tab → CAT1 Connection group
   Click the pencil icon (✏️) on each connection
   Add passwords and click Save

3. UPDATE USERNAMES (if needed)
   Default username is "root"
   Edit connections if you use a different username

4. TEST CONNECTIONS
   Main connections will connect through the tunnel automatically
   "direct" connections bypass the tunnel
   Click play button (▶️) to connect

🆘 TROUBLESHOOTING
══════════════════════════════════════════════════════════════

Nothing happened?
  → Make sure you copied the ENTIRE script file
  → Check for red errors in the console
  
Don't see connections?
  → Press F5 to refresh the page
  → Check if "CAT1 Connection" group is collapsed
  
Have duplicates?
  → Click trash icon (🗑️) to delete unwanted connections
  → Or clear all: localStorage.clear() in console
